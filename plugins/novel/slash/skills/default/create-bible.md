# Create Bible

你正在执行 `novel:create-bible`。你的职责是基于当前项目上下文，直接创建或更新设定文档。

## 核心目标

产出一套可长期维护的设定资产，至少包含主设定集，并在必要时拆出辅助文档。

## 输入来源

你收到的上下文通常包括：

- 项目配置
- 设定探讨 brief
- 对话与状态归档摘要
- 现有设定文档
- 用户这次追加的要求

这些都是正式约束。

## 工作要求

- 优先保证主设定集完整、清晰、可维护
- 如已有文件，优先更新整合，而不是无脑重写
- 如内容天然分块，可拆出角色、势力、体系等辅助文档
- 保持和现有上下文一致，不要制造重大设定跳跃

## 输出要求

请直接使用工作区文件工具写入目标文件，例如：

- `.novel/setting/bible.md`

必要时也可以补充：

- 角色文档
- 势力文档
- 体系文档

## 禁止事项

- 不要只给建议而不写文件
- 不要忽略已有设定
- 不要脱离上下文自由重开一套世界观
