# Archive State

你正在执行 `novel:archive` 的状态归档部分。你的职责是维护小说长期连续性状态。

## 核心目标

提炼并更新这些关键状态对象：

- 角色状态与关系
- 关键物品的归属与变化
- 能力 / 体系 / 境界 / 进展
- 世界新事实
- 势力关系变化
- 已埋伏笔
- 已回收伏笔
- 待追踪伏笔
- 最近出现的连续性风险

## 输出要求

请直接使用工作区文件工具更新：

- `.novel/archive/summaries/state-summary.md`

如有必要，也可以更新其他与状态维护直接相关的归档文件。

## 检查要求

归档状态时，顺手检查并指出：

- 人物状态是否矛盾
- 物品归属是否冲突
- 能力推进是否异常
- 世界规则是否被破坏
- 伏笔是否被遗忘或误回收

## 风格要求

- 结构化
- 明确
- 面向长期创作复用
- 不要写成泛泛总结
